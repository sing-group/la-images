Main user interface

The main user interface of LA-iMageS is organized into three main sections: the toolbar, the clipboard, and the analysis viewer. In the following figure, you can see the general aspect of LA-iMageS and the location of each section.

LA-iMageS main window

(click in the image to enlarge)

Through the toolbar, you will be able to access the main functions of LA-iMageS, where the most important is the data analysis operation. On the clipboard tree, you will find a list with the data loaded. Finally, you will be able to explore your data through the analysis viewer.
