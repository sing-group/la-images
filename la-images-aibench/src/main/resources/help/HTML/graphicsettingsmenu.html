The Graphic Settings menu

The Graphic Settings menu allows you to:

  • Change the Interpolation Level. By default None is selected, which means that no interpolation is applied.
  • Change the Color Map used to render the data.
  • Change the Range Mode. This option allows you to change the color map range, which can be: Element, so that the element range is used; Dataset, so that the dataset range is used; and Custom, so that you can use a custom range.
  • Show/hide the Color Bar Legend.
  • Show/Hide the Axes.
  • Show/Hide the Axes tick lines.
  • Reset configuration to default settings.
