Exploring data

Once the analysis execution finishes, a new analysis result will appear in the clipboard and in the Analysis viewer panel, as you can see in the following figure. Although the dataset will be automatically showed in the Analysis viewer panel once it has been loaded, at any time you can click on an analysis dataset in the clipboard to access its view.
